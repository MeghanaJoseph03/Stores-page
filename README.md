# Shopping Stores Landing Page

A responsive shopping stores landing page developed using HTML, CSS, and Bootstrap.

## Features
- Clean and modern user interface
- Displays popular shopping platforms
- Store logos with descriptions

## Technologies Used
- HTML5
- CSS3
- Bootstrap 4
- Flexbox

## Stores Included
- Amazon
- IKEA
- Meesho
- Flipkart
- Ajio
- Myntra
